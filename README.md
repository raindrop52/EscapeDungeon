# EscapeDungeon

장르 : 2D 퍼즐형 던전 탈출 게임

게임 소개 

던전에 갇힌 캐릭터를 탈출시켜라.

탈출을 막는 다양한 함정들이 존재하니 이를 피해 탈출할 수 있도록 해라.


조작 방법

pc : 방향키, 스페이스바

모바일 : 터치
